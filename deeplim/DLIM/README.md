# Deep LIMS in PyTorch


Deep LIMs in PyTorch
====

PyTorch implementation of Deep Linear Inverse Models (dLIMs) for regression based tasks.

## Requirements

  * PyTorch 0.4 or 0.5
  * Python 2.7 or 3.6

## Usage

```python train.py```
